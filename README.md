# Shuai Li academic homepage

This is a starter GitHub Pages site generated from a CV. It is a static website, so it only needs `index.html` and `style.css`.

## Publish on GitHub Pages

1. Create a GitHub repository named `YOUR-GITHUB-USERNAME.github.io`.
2. Upload `index.html` and `style.css` to the repository root.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch `main` and folder `/root`, then save.
6. Visit `https://YOUR-GITHUB-USERNAME.github.io` after GitHub finishes deploying.

## CV PDF

The generated page does not include the original CV PDF because it contains personal contact details. To add a public CV:

1. Create an `assets` folder.
2. Add a redacted file at `assets/Shuai_Li_CV.pdf`.
3. In `index.html`, uncomment the `Download CV` link in the hero section.

## Editing

- Update text directly in `index.html`.
- Update colors and spacing in `style.css`.
- Add a headshot by saving an image in `assets/` and inserting an `<img>` tag in the hero section.
